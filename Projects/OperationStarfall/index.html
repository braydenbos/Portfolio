Operation starfall

The Project

The goal of Operation Starfall is to make a multiplayer couch couch co-opin the retro 2.5D style from the 80's. This project started in september 2021 but I joined september 2023. This project is being made in the unity engine and mainly coded with C#. This project was a part of my third period om my third year on mediacollege amsterdam as an hybrid learning environment.

My Part

Loot system

In this project I worked on the loot system and made a loot simulator.
The loot system was verly easy to edit I first made a new singelton that would do all the maths and loot dropping.
Then I made a new scriptable object that would hold the force at wich it should drop the loot so it would be easaly editable.

Loot simulator

After finishing the loot system I made a loot sim. I did so with custom editors and GUI.
In the custom edator I made a button that would activate a function in the system and roll the loot drop with the given loot table.
That I used to genarate a loot drop and put the drops in a list to make it more visual.
Than I worked on a system that would count all items with the same name and put them together to make it easier to read.
After that I made a slider that would make it possable to roll multable simulations at onece.
And lastly I made the simulation work while the game was running to also show the loot in game.
